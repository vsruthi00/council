# Theme: Soft Pastel

## When to use

Consumer apps, lifestyle products, wellness tools, and friendly onboarding flows. Use when the audience skews toward non-technical users and approachability matters more than density or seriousness.

## Color tokens

Token values extracted from `.t-pastel` block in preview.html.

| Token     | Hex       | Notes                                               |
|-----------|-----------|-----------------------------------------------------|
| bg        | #f7f5ff   | Light lavender page background                      |
| surface   | #ffffff   | Input and card background                           |
| border    | #ece8fb   | Input border, card border, badge background         |
| text      | #2e2a3f   | Primary body text (cool near-black)                 |
| muted     | #7a7490   | Secondary text, lede, card body                     |
| accent    | #8b7cf0   | Primary button, focus ring                          |
| secondary | #e8fbf2   | Secondary button background (mint); secondary button text #2e8b67, border #c8f0dd |

Additional observed value: badge text `#6a5acd`, swatch shows `#5fcf9e` (mint accent used in secondary button).

## Fonts

- Headings: Nunito, sans-serif (weight 700)
- Buttons: Nunito, sans-serif
- Body: Inter, sans-serif (inherited)

## Radius

14px for inputs, buttons, and cards. 12px for badges.

## Motion / depth note

No shadows or glows; large radius and pastel palette alone create a friendly, low-tension feel.
