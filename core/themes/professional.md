# Theme: Professional

## When to use

Business software, fintech, enterprise SaaS, legal or compliance tools, and any product where the design must communicate reliability and seriousness before anything else.

## Color tokens

Token values extracted from `.t-pro` block in preview.html.

| Token     | Hex       | Notes                                      |
|-----------|-----------|--------------------------------------------|
| bg        | #ffffff   | Main page / demo background                |
| surface   | #f4f6f8   | Card and mini-card background              |
| border    | #d8dee6   | Input border, card border                  |
| text      | #14233a   | Primary body text (dark navy)              |
| muted     | #5a6b80   | Secondary text, labels, lede, card body    |
| accent    | #1f5fa8   | Primary button, focus ring, badge text     |
| secondary | #e7eff8   | Badge background                           |

## Fonts

- Headings: Inter, sans-serif (weight 700)
- Body: Inter, sans-serif

## Radius

4px for inputs, buttons, and cards. 3px for badges (minimal rounding signals structure over friendliness).

## Motion / depth note

No glow or elevation; tight radius and restrained palette carry the authority signal without decoration.
