# Theme: Natural

## When to use

Wellness apps, ecology or sustainability products, education platforms, lifestyle services, and any interface that should feel grounded, organic, and unhurried.

## Color tokens

Token values extracted from `.t-natural` block in preview.html.

| Token     | Hex       | Notes                                                   |
|-----------|-----------|---------------------------------------------------------|
| bg        | #f4f1e9   | Oat/linen page background                              |
| surface   | #fbfaf4   | Input and card background                              |
| border    | #e2ddd0   | Input border, card border                              |
| text      | #2c352c   | Primary body text (dark forest green-black)             |
| muted     | #6e7468   | Secondary text, lede, card body                        |
| accent    | #4f7a5a   | Primary button, focus ring, badge text (`#3f6349`)      |
| secondary | #8a5a34   | Secondary button text (warm clay); button border `#ddd0bd` |

Additional observed value: badge background `#e4ebe0`; label color `#7a8270`.

## Fonts

- Headings: Inter, sans-serif (weight 600)
- Body: Inter, sans-serif

## Radius

10px for inputs, buttons, and cards. 8px for badges.

## Motion / depth note

No shadows or glows; the earthy palette and medium radius provide calm without any motion or depth layering.
