# Theme: Warm Editorial

## When to use

Content-forward products: blogs, newsletters, education platforms, and reading apps. Best when long-form text is the primary interface and the design should feel welcoming rather than utilitarian.

## Color tokens

Token values extracted from `.t-warm` block in preview.html.

| Token     | Hex       | Notes                                      |
|-----------|-----------|--------------------------------------------|
| bg        | #faf6f0   | Warm paper background                      |
| surface   | #fffdfa   | Input and card background                  |
| border    | #e7ddd0   | Input border, card border                  |
| text      | #2b2520   | Primary body text (warm near-black)        |
| muted     | #6b6258   | Secondary text, lede, card body            |
| accent    | #b85c38   | Primary button, focus ring, badge text     |
| secondary | #f3e6dc   | Badge background                           |

Additional observed value: label color `#8a7d6d` (lighter muted, used for form labels); secondary button border `#d8cbb8`.

## Fonts

- Headings: Newsreader, Georgia, serif (weight 600, size 23px)
- Body / lede: Newsreader, Georgia, serif
- Labels and UI chrome: Inter, sans-serif (inherited from body)

## Radius

6px for inputs, primary and secondary buttons, and cards. 4px for badges.

## Motion / depth note

No glow or elevation effects; warmth comes from color temperature alone, keeping interactions calm and unhurried.
